# yolo-object-detection
Обнаружение объектов с помощью YOLOv8

О проекте

Цель проекта — обучить модель для автоматического обнаружения объектов на изображениях и протестировать качество детекции.
В этом проекте реализована модель для обнаружения объектов на изображениях с использованием YOLOv8.
Проект выполнен на основе опыта стажировки в области искусственного интеллекта и компьютерного зрения.
В ходе работы были обучены и протестированы модели YOLOv8 и YOLOv5, а также собран базовый pipeline для обучения и применения модели.

Возможности

•	обучение модели YOLOv8 на пользовательском датасете
•	запуск инференса (предсказаний) на изображениях
•	визуализация результатов детекции

Стек технологий

•	Python
•	Ultralytics YOLOv8
•	OpenCV
•	NumPy
•	Google Colab

Обучение модели

•	модель: YOLOv8
•	обучение проводилось в Google Colab
•	использовался датасет в формате YOLO
•	размер изображений: 640*640
•	количество эпох задаётся параметрами

Как запустить проект

Установка зависимостей:
pip install -r requirements.txt

Обучение модели:

python src/train.py
Запуск инференса:
python src/predict.py

Результаты

- обучена модель YOLOv8  
- проведен инференс на новых изображениях  
- визуализированы результаты
